# Student Management System (Python)

## Project Description
This is a simple Python program created while learning Python dictionaries.
The program allows the user to add, view, update, and delete student records
using a menu-driven approach.

## Features
- Add student name and marks
- Display all student details
- Update student marks
- Delete student record
- Exit the program safely

## How the Program Works
- User selects an option from the menu
- Program performs the selected operation
- Data is stored using a dictionary
- Menu repeats until user exits

## Concepts Used
- Dictionary
- While loop
- Conditional statements
- User input handling
- Menu-driven programming

## Learning Outcomes
- Learned how to use dictionaries in Python
- Understood CRUD operations (Create, Read, Update, Delete)
- Improved logical thinking
- Practiced menu-based programs

## How to Run
1. Open terminal or command prompt
2. Run the program using: python student_management.py
